
# 📘 ML Foundations — Week 1 Assignment

## 📌 Overview
A comprehensive week 1 assignment for Machine Learning Foundations covering essential mathematical and programming concepts required for ML. This notebook includes hands-on implementations, exercises, and visualizations across 6 core topics.

## 🎯 Topics Covered
| Topic | Key Concepts |
|-------|--------------|
| **Python Fundamentals** | Data types, control flow, data structures, exceptions, functions, lambdas |
| **NumPy** | Array creation, shapes, indexing, slicing, operations, dot product |
| **Pandas** | DataFrames vs Series, iloc/loc, filtering, group by, missing data handling |
| **Linear Algebra** | Vectors, matrices, eigenvalues, eigenvectors, SVD, dimensionality reduction |
| **Statistics** | Descriptive/inferential statistics, hypothesis testing, error metrics, distribution testing |
| **Probability Theory** | Core concepts, distributions, Bayes' theorem, Central Limit Theorem |

## 📋 Assignment Structure
### Part 1 — Python Fundamentals
- Data Types & Control Flow (`classify_number`)
- Data Structures (word counting, unique words, list comprehensions)
- Exceptions (`safe_divide`)
- Functions & Lambdas (`apply_twice`)

### Part 2 — NumPy
- Array Creation & Shapes (1D, 2D, 3D arrays)
- Indexing & Slicing (rows, columns, submatrices, boolean indexing)
- Operations & Dot Product (element-wise, matrix multiplication)

### Part 3 — Pandas
- DataFrames vs Series extraction
- iloc & loc selection
- Filtering & Group By operations
- Missing data handling (fill, drop)

### Part 4 — Linear Algebra
- Vectors & Matrices as Data (L2 norm, visualization)
- Matrix Operations (addition, scalar multiplication, multiplication)
- Eigenvalues & Eigenvectors (computation, verification, plotting)
- SVD & Dimensionality Reduction (reconstruction, rank-1 approximation)

### Part 5 — Statistics
- Descriptive vs Inferential Statistics (mean, median, std, IQR)
- Hypothesis Testing (t-tests, p-value interpretation)
- Error Metrics (MAE, MSE, RMSE, R², Adjusted R²)
- Distribution Testing & Stationarity (KS test, ADF test)

### Part 6 — Probability Theory
- Core Concepts (sample space, joint/conditional probability)
- Distributions in the Wild (Normal, Binomial, Poisson with ML use cases)
- Bayes' Theorem (spam filter scenario, naive Bayes predictor)
- Central Limit Theorem (sample means distribution, CLT validation)
